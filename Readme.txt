1、心臟瓣膜置換術後康復期應適當限制主食、鹽、糖及脂肪，宜少量多餐手段以減輕心臟負擔；在三餐以外，另加少量點心、水果等。

2、飲食宜給予平淡、多進含豐富蛋白質、維生素及高熱量易消化飲食，如魚、肉、蛋、奶等；少量多餐，多給蔬菜和水果；心功能不全者給低鹽飲食，並限制水分攝入。

3、因菠菜、白菜、新鮮豌豆、菜花、豬肝等含維生素K較多，在換瓣術後不宜多吃（維生素K可對抗抗凝藥物，縮短凝血酶原時間）。


心臟瓣膜病的症狀

1、瓣膜病的病人主要是無緣無故出現乏力，本來平常能幹一定強度的活兒，現在可能勝任不了，容易累。

2、氣短，一幹活兒就心慌，上不來氣;也有的人出現腿腫。

3、食欲不振，也就是說一段時間吃飯不好，胃腸道瘀血可能消化不好，出現肚子脹。

4、二尖瓣膜聽診區可聞舒張期隆隆樣雜音或收縮吹風樣雜音，主動脈聽診區可聞收縮期或舒張期樣雜音。

5、肝大、肝頸靜脈回流徵陽性、腹水等。



-------
心臟瓣膜病是我國一種常見的心臟病，其中以風濕熱導致的瓣膜損害較好為常見。那麼對於這樣的一種心血管疾病，在手術之後患者該如何飲食呢？

1、嚴格限制鹽的攝入量

    鈉鹽對身體的水分具有瀦留作用，會加重心臟的負擔，出現水腫等，嚴重時造成心力衰竭等晚期風心病症狀。因而要嚴格限制鹽的攝入量，每天少吃各種鹽製品，鹽要控制在5克以下。
 
2、禁止食用苦寒和辛辣食品

    辣椒、芥末等食品刺激性大，會加快人的心跳，增加心臟負擔，此類食品多吃還容易造成便秘，導致排便困難，不僅加重心臟負擔，而且可能發生不測。吃苦寒食品會加重病情。

3、戒菸、酒、濃茶和咖啡

    風心病患者嚴禁吸菸、喝濃茶和咖啡。煙燃燒產生大量的一氧化碳，當一氧化塘吸入體內後，會造成全身血管收縮，並與血中的血紅蛋白結合，致使其輸送氧氣的功能下降，造成心肌缺血，這對本已有病變的患者更為不利。而酒、濃茶和咖啡等興奮刺激性的飲料，會造成血壓升高，增強神經系統的興奮性，加快心率，進而誘發心律紊亂，加重心臟負擔，造成心悸瓣膜的進一步損害。因而，風心病患者應禁菸、酒等興奮刺激性飲料。

4、限制熱量供給

    風濕性心臟病患者熱量攝入一般每日每公斤體重25～35卡，身體肥胖者可按下限供給，有助於減輕心臟負擔。

5、限制高脂肪、高膽固醇攝入

    風濕性心臟病患者心臟功能存在不同程度的受損，宜限制高脂肪、高膽固醇攝入，以免消化不良，儘量避免吃蛋黃、動物內臟、動物油、螃蟹、魚子等。


 
總之，心臟瓣膜病人大多伴有不同程度的心功能不全，平時應注意少吃哪些容易加重心衰、水腫的食物，如嚴格限制食鹽攝入，少喝水、湯。做完瓣膜置換手術後應注意不吃那些易妨礙抗凝藥物效果的食物。此外術後3個月內，原則上病人應充分休息和靜養。根據患者體力情況有規律地安排作息時間，進行適當室內和室外活動，但要量力而行，定期複查。
